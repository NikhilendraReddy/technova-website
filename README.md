# TechNova Solutions Pvt. Ltd. — Corporate Website

## Overview

This project is a responsive multi-page website developed for a service-based IT company, **TechNova Solutions Pvt. Ltd.** The website is designed to present the company’s services, mission, and contact information in a clean, modern, and professional layout.

The primary objective of this project is to demonstrate frontend development skills, including responsive design, user interface structuring, and basic interactivity using JavaScript.

---

## Features

* Fully responsive design (mobile, tablet, and desktop)
* Modern and professional user interface
* Multi-page navigation (Home, About, Services, Contact)
* Contact form with client-side validation
* Dark and light mode toggle
* Smooth layout and consistent styling
* Organized folder structure for scalability

---

## Pages Included

1. **Home**

   * Company introduction
   * Hero section with call-to-action
   * Key features and testimonials

2. **About**

   * Company overview
   * Mission and vision

3. **Services**

   * Detailed overview of IT services offered

4. **Contact**

   * Contact form with validation
   * Company contact details
